# RNN 相关代码

## RNNPredictRequest 

一个用LSTM预测HTTP请求数的简单例子

数据格式如下：

| timestamp     | count |
| ------------- | ----- |
| 1485773733123 | 1139  |
| 1485773734123 | 1760  |
| ...           | ...   |

这里只使用count属性，构建训练数据、测试数据

使用前t秒数据，预测下一秒数据

结构如下：

```python
X:
[[d1, d2, d3, ... dt-1]
...
]
Y:
[dt,...]
```
